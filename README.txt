Projeto2023.1:
Main: Unir comandos criados, criar menus para o usuário escolher qual a função que ele deseja e criar comandos de
validação de entradas.
P1: Criar comando para informar nota mínima que o usuário precisa tirar na P2, P3 e P4 para passar no período.
P2: Criar comando para informar nota mínima que o usuário precisa tirar na P3 e P4 passar no período e nota mínima
que precisa tirar na P3 e P4 para ir pra AF e passar na AF.
P3: Criar comando para informar nota mínima que o usuário precisa tirar P4 passar no período e nota mínima que
precisa tirar na P4 para ir pra AF e passar na AF.
P4: Verificar se o aluno passou e, caso ele não tenha passado, informar a nota que ele precisa tirar para
passar na AF.

Project2023.1:
Main: Unite created commands, create menus for the user to choose which function he wants and create commands of
input validation.
P1: Create a command to inform the minimum grade that the user needs to get in P2, P3 and P4 to pass the period.
P2: Create command to inform the minimum grade that the user needs to take in P3 and P4 to pass the period and minimum grade
that needs to be removed in P3 and P4 to go to AF and pass in AF.
P3: Create command to inform minimum grade that the user needs to take P4 pass in the period and minimum grade that
need to take P4 to go to AF and pass in AF.
P4: Check if the student has passed and, if he has not, inform the grade he needs to get to
pass in AF.
